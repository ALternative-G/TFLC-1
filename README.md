# TFLC-1: Разработка GUI для языкового процессора

---

## Описание проекта
Проект представляет собой текстовый редактор с графическим интерфейсом приспособленым для будущего языкового процессора с модулями сканера, парсера и потенциалом для интеграции других.

На текущем этапе реализован полноценный пользовательский интерфейс с возможностью работы с несколькими файлами одновременно, настраиваемым отображением и полной локализацией на русском и английском языках.

## Интерфейс программы
Скриншоты работы программы можно посмотреть на вики

### Основные элементы интерфейса:
- **Главное меню** - полный набор функций для работы с файлами и текстом
- **Панель инструментов** - быстрый доступ к основным операциям
- **Система вкладок** - работа с несколькими файлами для ввода и программами для вывода одновременно
- **Панель нумерации строк** - автоматическая нумерация для каждого файла
- **Строка состояния** - отображение текущего статуса и информации

## Инструкция по установке и запуску
1. Перейдите в папку `bin\Release` или `bin\Debug`
2. Запустите файл `TFLC_GUI.exe` двойным щелчком мыши


### Системные требования:
- Запуск релизного файла должен быть возможен без предустановки IDE и иных компонентов

## Руководство пользователя

### Меню "Файл"
| Функция | Горячие клавиши | Описание |
|:---|:---:|:---|
| **Создать** | `Ctrl` + `N` | Создает новый пустой файл в текущей вкладке |
| **Новая вкладка** | `Ctrl` + `Shift` + `T` | Создает новую пустую вкладку |
| **Открыть** | `Ctrl` + `O` | Открывает диалог выбора файла(ов) |
| **Сохранить** | `Ctrl` + `S` | Сохраняет текущие изменения в открытом файле |
| **Сохранить как** | — | Сохраняет текущий текст в новый файл |
| **Закрыть вкладку** | `Ctrl` + `W` | Закрывает текущую вкладку |
| **Выход** | `Alt` + `F4` | Закрывает программу (с проверкой всех вкладок) |

### Меню "Правка" и Горячие клавиши

| Функция | Горячие клавиши | Описание |
|:---|:---:|:---|
| **Отменить** | `Ctrl` + `Z` | Отменяет последнее действие |
| **Повторить** | `Ctrl` + `Y` | Повторяет ранее отмененное действие |
| **Вырезать** | `Ctrl` + `X` | Удаляет выделенный текст и помещает его в буфер обмена |
| **Копировать** | `Ctrl` + `C` | Копирует выделенный текст в буфер обмена |
| **Вставить** | `Ctrl` + `V` | Вставляет текст из буфера обмена |
| **Удалить** | `Ctrl` + `D` | Удаляет выделенный текст без сохранения в буфер |
| **Выделить всё** | `Ctrl` + `A` | Выделяет весь текст в активной вкладке |

### Меню "Вид"
| Функция | Описание |
|:---|:---|
| **Увеличить шрифт** | Увеличивает размер шрифта (до 72pt) |
| **Уменьшить шрифт** | Уменьшает размер шрифта (до 6pt) |
| **Сбросить шрифт** | Возвращает стандартный размер шрифта (10pt) |

### Меню "Локализация"
| Функция | Описание |
|:---|:---|
| **Русский** | Переключает интерфейс на русский язык |
| **English** | Переключает интерфейс на английский язык |

### Меню "Текст" (Шаблонные разделы)
- Постановка задачи
- Грамматика
- Классификация грамматики
- Метод анализа
- Тестовый пример
- Список литературы
- Исходный код программы

### Меню "Справка"
| Функция | Описание |
|:---|:---|
| **Вызов справки** | Вызывает окно с подробным руководством пользователя |
| **О программе** | Информация о разработчике и версии ПО |

## Дополнительные возможности

### Drag & Drop поддержка
- Возможность перетащить файлы в окно программы для быстрого открытия
- `Ctrl` + перетаскивание - автоматическое открытие в новой вкладке
- Поддержка множественного выбора файлов

### Подсветка синтаксиса
Подсветка ключевых слов языков программирования (при нажатии опции меню "пуск"):
- Управляющие конструкции (`if`, `for`, `while`, `switch`)
- Модификаторы доступа (`public`, `private`, `protected`)
- Типы данных (`int`, `string`, `bool`, `float`, `double`)

### Нумерация строк
- Автоматическая нумерация для каждой вкладки
- Синхронизация прокрутки с основным текстом

### Работа с несколькими файлами
- Каждая вкладка сохраняет свою историю отмены/повтора
- Индикатор изменений (*) для модифицированных файлов

### Панели результатов
- **Сканер** - таблица для вывода результатов лексического анализа
- **Парсер** - таблица для вывода результатов синтаксического анализа
- **Оптимизация** - таблица для вывода результатов оптимизации

## Текущий статус разработки

### Реализовано:
- Основное меню программы.
- Панель инструментов с кнопками для быстрого доступа к часто используемым функциям.
- Область ввода/редактирования текста (текстовый редактор).
- Область отображения результатов работы языкового процессора (только для вывода, без возможности редактирования).
- Пользователь имеет возможность изменять размеры оконных областей и всего приложения. При необходимости автоматически появляются полосы прокрутки.
- Меню "Файл": Создание, открытие, сохранение, сохранение как, выход (с подтверждением сохранения изменений).
- Меню "Правка": Стандартные операции с текстом: отмена/возврат, вырезание, копирование, вставка, удаление, выделение всего текста.
- Меню "Справка": Вызов справочной системы (на данном этапе – описание реализованных функций) и окна "О программе".
- Все функции из меню "Файл", "Правка" и "Справка" доступны через соответствующие кнопки на панели инструментов.
- Приложение самодостато и должно запускаться на целевой машине без установленной среды разработки (IDE) и библиотек.
- Интерфейс с вкладками, позволяющий одновременно работать с несколькими текстами (для окна редактирования).
- Выбор языка интерфейса приложения (интернационализация).
- Нумерация строк в окне редактирования текста.
- Открытие файла при перетаскивании иконки в окно программы.
- Наличие строки состояния для отображения текущей информации о состоянии работы приложения.
- Базовая подсветка синтаксиса в окне редактирования.
- Интерфейс с вкладками, позволяющий работать с разными модулями программы (для окна вывода результатов)
- Отображение ошибок в окне вывода результатов в виде таблицы.
- Горячие клавиши для быстрых команд.
